Phần này hướng dẫn bạn khởi tạo dự án AI Agent từ template AI20K chỉ trong vài bước đơn giản. Bạn sẽ học cách clone repository, cấu hình biến môi trường, xác nhận hệ thống hoạt động đúng và thiết lập Git workflow. Đây là bước đầu tiên và quan trọng nhất trước khi bắt tay vào code. Làm đúng từ đầu sẽ giúp bạn tiết kiệm rất nhiều thời gian sửa lỗi sau này.
