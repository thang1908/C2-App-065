Phần này trình bày tổng quan kiến trúc hệ thống của một dự án AI Agent hoàn chỉnh. Bạn sẽ hiểu được các thành phần chính, cách chúng giao tiếp với nhau và luồng dữ liệu đi qua hệ thống. Các sơ đồ Mermaid giúp bạn hình dung trực quan kiến trúc trước khi đi vào code. Nắm vững kiến trúc là nền tảng để xây dựng agent có thể mở rộng và bảo trì dễ dàng.
