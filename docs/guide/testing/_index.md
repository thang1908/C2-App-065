Phần này hướng dẫn cách viết test cho dự án AI Agent một cách bài bản và hiệu quả. Bạn sẽ học cấu trúc thư mục test, cách viết test cho API endpoints và agent logic, cũng như sử dụng conftest.py để quản lý fixtures. BTC yêu cầu tối thiểu một số lượng test nhất định — viết test tốt không chỉ giúp đạt điểm cao mà còn đảm bảo agent hoạt động ổn định.
